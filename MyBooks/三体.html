                              # 序言

                              人类之所以能够超越地球上的其它物种建立文明，主要是因为他们能够在自己的大脑中创造出现实不存在的东西，在未来，当人工智能拥有超过人类智力时，想象力也许是我们对于它们所拥有的惟一优势。

                              大部分人的人生都是偶然，是的，整个人类历史也是偶然，从石器时代到今天都没什么重大突破，真幸运。但既然是幸运总有结束的一天。

                              # 第三章 射手和农场主

                              射手假说：有一名神枪手，在一个靶子上每隔十厘米打一个洞。设想这个靶子的平面上生活着一种二维智能生物，它们中的科学家在对自己的宇宙进行观察后，发现了一个伟大的定律："宇宙每隔10厘米，必然会有一个洞。"它们把这个神枪手一时兴起的随意行为，看成了自己宇宙中的铁律。
                              农场主假说：一个农场里有一群火鸡，农场主每天中午11点来给它们喂食。火鸡中的一名科学家观察这个现象一年了都没有例外，于是它也发现了一个伟大的定律："每天中午11点，就会有食欲降临。"它在感恩节早晨向火鸡们公布了这个定律，但这天上午11点没有食物降临，农场主进来把它们都捉去杀了。

                              人类存在的时间只是200万年左右，宇宙存在的时间为140亿年左右，相比而言，人类科技发展时间在宇宙中的尺度是非常微小的。迄今为止人类的一切认知均建立在自己文明的经验学说之上。我们没有足够具有说服力的证据证明人类认为的科学是这个庞大宇宙中的真理。这是人类认知上的局限性，文明的等级差距在这两个假说上体现得淋漓尽致。我们是否又只是一种三维智能生物亦或者是火鸡呢？

                              # 第五章 叶文洁

                              有两幅图。一幅是画面庞杂精细的《清明上河图》，另一幅是一张空旷的天空照片，空荡荡的蓝天上只有一缕似有似无的白云。教授问这两幅画中哪一幅所包含的信息量更大，答案是后者要比前者大一至两个数量级！

                              如何理解呢?先看看下面这个故事!

                              最孤独的人

                              1969年7月20日，迈克尔科林斯突然成了全世界最孤独的人。

                              科林斯是执行"阿波罗11号"登月任务的三名美国宇航员之一，他的任务是呆在"哥伦比亚号"指令舱中绕月球飞行，直到指令长尼尔•阿姆斯特朗和登月舱驾驶员巴兹•奥尔德林完成任务归来。

                              这意味着科林斯无法成为首批登陆月球的人，他的名字不会像那两位同伴一样被铭记。

                              当阿姆斯特朗和奥尔德林乘坐的"鹰号"登月舱旋转着飞向月球表面着陆点时，科林斯抓起手边的相机，拍下了一张照片。

                              照片背景中那颗小小的蓝色星球上，有6.5亿人正在观看登月直播，他们即将见证阿姆斯特朗代表人类在太空探索史上迈出的一大步。

                              科林斯当时不曾想到，当他按下快门那一刻，便留下了一张全世界古往今来所有人的大合影，只有他自己不在其中。

                              根据质星守恒定律，无论是在这张照片拍摄前去世的人，还是在它拍摄后出生的人，构成他们的物质都被收罗在这张照片中，只是实体形式有可能发生变化，这就是所谓的质量既不能被创造也不能被破坏。

                              这是科林斯第二次也是最后一次太空任务，他独自在"哥伦比亚号"上等待了21个半小时，每当指令舱飞到月球背面，他和地球的通讯都要中断48分钟。在漫长的等待中，科林斯除了清理维护机舱等待两名队友的回归，剩下的时间都在孤独煎熬。

                              "我并不否认有孤独的感觉"，科林斯后来说，"特别是和地球中断联系的时候，那种孤独感格外强烈。我是真正的孤身一人，与任何已知的生命完全隔绝，月球那边是36亿人，月球这边只有我自己。"

                              柯林斯在2021年4月28日以90岁高龄去世，他在去世前几年接受采访时曾示，当时他非常担心阿姆斯特朗和奥尔德林的安全，如果他们不幸死在月球上，那么他将独自返航。

                              假如那样的悲剧发生，科林斯作为唯一幸存者必将抱憾终身，但同时他的名字也会举世皆知，而不是被媒体戏称为"被遗忘的宇航员"。

                              # 第六章 红岸之二

                              "西方资产阶级的奢侈玩意儿，用微波被吸收后产生的热效应加热食物。我以前在的那个研究所，为了精密测试某种元件的高温老化，从国外进口了一台。我们下了班也用它热馒头、烤土豆。很有意思，里面先热，外头还是凉的。"雷政委说着站了起来，来回踱步，他走得如此贴近悬崖边缘，令叶文洁十分紧张，"红岸系统就是一台微波炉，加热的目标是敌人在太空中的航天器。只要达到0.1～1瓦/平方厘米的微波能量辐射，就可直接使卫星通信、雷达、导航等系统的微波电子设备失效或烧毁。"

                              微波炉热东西都是从里到外热的吗？其实不尽然，基本来说微波炉是以加快分子运动的方式来加热的，其中对水分子的效果最强。微波炉加热的范围跟水分子的分布很有关系，如果有很多水分子在内部，那么的确是从里面开始加热的，如果有很多水分子在外面那么很多能量会被这些水分子吸收。传递到内部已经没有多少能量了。而且微波一般来说只能穿透两公分左右的食物，如果物体过大，很有可能只有两公分的表皮部分被加热，剩下的中心部分需要热传导来加热。

                              # 第七章 三体问题

                              蒙特卡洛法

                              是一种计算不规则图形面积的计算机程序算法，具体做法是在软件中用大量的小球随机击打那块不规则图形，被击中的地方不再重复打击，这样，达到一定的数量后，图形的所有部分就会都被击中一次，这时统计图形区域内小球的数量，就得到了图形的面积，当然，球越小结果越精确。这种方法虽然简单，却展示了数学中的一种用随机的蛮力对抗精确逻辑的思想方法，一种用数量得到质量的计算思想。

                              洛希极限

                              法国天文学家洛希证明，任何坚固的天体，在接近另一个比它大得多的天体的时候，都会受到强大的潮汐力作用而最终被扯成碎片。这个较小的天体会被拉碎的距离称为洛希极限。通常是大天体赤道半径的2.44倍。

                              奥林匹斯山

                              位于火星北半球的一个盾状火山，高大约25千米，为火星最高的山峰，是太阳系中已发现的最高火山。宽700千米，火山口直径为80千米。在希腊神话中，奥林匹斯山是众神的住所。

                              # 第八章 智子

                              请看以下事实：人类从狩猎时代到农业时代，用了十几万地球年；从农业时代到工业时代用了几千地球年；而由工业时代到原子时代，只用了二百地球年；之后，仅用了几十个地球年，他们就进入了信息时代。这个文明，据有可怕的加速进化能力！而在三体世界，已经存在过的包括我们在内的二百个文明中，没有一个经历过这种加速发展，所有的三体文明的科学和技术的进步都是匀速甚至减速的。我们世界的各个技术时代，都需要基本相同的漫长发展时间。

                              # 第九章 虫子

                              你们是虫子！
                              对，我们是虫子。看看吧，这就是虫子，它们的技术与我们的差距，远大于我们与三体文明的差距。人类竭尽全力消灭它们，用尽各种毒剂，用飞机喷撒，引进和培养它们的天敌，搜寻并毁掉它们的卵，用基因改造使它们绝育；用火烧它们，用水淹它们，每个家庭都有对付它们的灭害灵，每个办公桌下都有像苍蝇拍这种击杀它们的武器……这场漫长的战争伴随着整个人类文明，现在仍然胜负未定，虫子并没有被灭绝，它们照样傲行于天地之间，它们的数量也并不比人类出现前少。把人类看做虫子的三体人似乎忘记了一个事实：虫子从来就没有被真正战胜过。

                              # 中部 咒语

                              光锥之内就是命运！光的传播沿时间轴呈锥状，物理学家们称为光锥，光锥之外的人不可能了解光锥内部发生的事件。想想现在，谁知道宇宙中有多少重大事件的信息正在以光速向我们飞来，有些可能已经飞了上亿年，但我们仍在这些事件的光锥之外。
