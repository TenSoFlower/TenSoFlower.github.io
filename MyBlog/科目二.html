                  # 科目二

                  # 直角转弯

                  对线行驶(雨刮器铆钉位置领右边行驶)

                  对线正确之后再开左转向灯

                  左门扣的尾端对角，向左打死方向盘

                  车正则回正

                  # 曲线行驶

                  挡风玻璃下角对左线行驶

                  前车盖左突起部分刚压线向左打一圈微调

                  前车盖左突起部分又压线时回正方向

                  雨刮器铆钉位置压线向右打一圈微调

                  车正则回正方向

                  # 侧方停车

                  对线行驶(雨刮器铆钉位置领右边行驶)前库角出现在右视镜中间停车

                  前库角在右视镜消失的瞬间向右打满方向

                  左视镜能看见库脚回正并且向左打180度

                  右轮压线向左打死(一圈)

                  车尾与线微窄停车(如果很宽则平行停车)

                  开左转向灯

                  前车盖左突起部分刚压线回正方向

                  雨刮器铆钉位置压线向右打一圈微调

                  车正则回正方向

                  # 倒车入库

                  挡风玻璃下角对左线行驶

                  座椅座在线上停车

                  倒车至左后视镜下沿遮虚线右打满方向

                  看右视镜，当车尾离右库脚小于30公分距离时则回一圈如果很宽则不动

                  车尾与线微窄则回正方向，后退一定距离停车，观察左后视镜和右后视镜如果左边车离线很宽则下述步骤倒车时需要向左微调

                  出库当左黄线到达前窗后下角时左打一圈加90度方向保持不动压住线停车

                  倒车时看情况需不需要微调，当车尾离右库脚小于30公分距离时则回一圈如果很宽则打死方向

                  车尾与线平行则回正方向，后退一定距离停车
