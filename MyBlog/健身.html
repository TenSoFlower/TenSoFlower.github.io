            # 体重指数

            BMI：体重(kg) ÷ 身高(m)的平方

            比如 BMI = 70 ÷ 1.78² = 22.1

            BMI < 18.5 低于正常体重

            18.5 <= BMI < 24 正常体重

            24 <= BMI < 28 超重

            BMI >= 28 肥胖
